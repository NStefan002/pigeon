# Pigeon

This is a plugin that shows different module status on the status line or winbar.

![](/image.png)
***
![](/image-thin.png)

## Modules

*   \[x] battery
    *   power status for a laptop
*   \[x] internet
    *   \[x] wifi connection
    *   \[ ] ethernet connection
    *   \[ ] internet connection speed
*   \[x] date
    *   Current date
*   \[x] time
    *   Current time
*   \[x] day
    *   Current day
*   \[ ] cpu
    *   System processor usage
*   \[ ] ram
    *   System memory usage
*   \[ ] updates
    *   Neovim plugins updates
